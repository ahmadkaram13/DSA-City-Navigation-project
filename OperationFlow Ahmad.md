# Operation Flow Explanation — City Navigation System

## 1. Operation: Find Shortest Path  
**Member:** Ahmad Karam

### What the operation does  
Finds the shortest possible route between two locations.

### Data Structure  
Priority Queue + Graph (Adjacency List)

### Why suitable  
Efficient for Dijkstra’s algorithm.

### User interaction  
User selects start + destination → system computes shortest path.
